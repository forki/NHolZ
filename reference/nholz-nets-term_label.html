term_label

Namespace: NHolZ
Parent Module: nets

Term nets are a finitely branching tree structure; at each level we have a set of branches and a set of "values". Linearization is performed from the left of a combination; even in iterated combinations we look at the head first. This is probably fastest, and anyway it's useful to allow our restricted second order matches: if the head is a variable then then whole term is treated as a variable.

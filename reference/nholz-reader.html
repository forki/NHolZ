NHolZ

Reader

Namespace: NHolZ

This module provides library support for generic reader functions that
process data from a given source and return a read item and the source
correspondingly advanced, ready for the next item to be read. This gets used in the HOL quotation lexer and parsers (see 'Lexer' and 'Parser'
modules respectively).

Nested types and modules

TypeDescription
ReadFail

This exception is dedicated to non-fatal failures in reader functions, and is used for simple control flow between reader connectives. This allows
simple implementation of reader connectives that trap non-fatal failures
and act accordingly, whilst allowing other exceptions to propagate.
